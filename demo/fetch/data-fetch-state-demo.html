data-fetch-state デモ

フェッチの状態（読み込み中・成功・失敗）を _fetch として参照し、 画面個別の JavaScript を書かずに data-if だけで表示を出し分けるデモです。

読み込み中...

読み込みに失敗しました（ステータス: {{_fetch?.statusCode}}）。再試行してください。

取得に成功しました。

  • {{name}}

「成功するフェッチ」または「失敗するフェッチ」を押してください。

_fetch の中身

注入される状態オブジェクトの構造:

{
  "status": "loading" | "success" | "error",
  "loading": boolean,
  "success": boolean,
  "error": boolean,
  "statusCode": number | null,
  "message": string | null
}

_fetch は表示制御用の内部状態で data-bind 属性には書き出されず、 送信ペイロードにも含まれません。自動リトライは行わず、再取得は data-click-fetch で手動導線を宣言します。

マークアップ例

<button
  data-click-fetch="./data-fetch-state-demo.json"
  data-click-bind="#result"
  data-click-fetch-state="#result"
>成功するフェッチ</button>

<div id="result" data-bind='{"items":[],"_fetch":null}'>
  <p data-if="_fetch?.loading">読み込み中...</p>
  <div data-if="_fetch?.error">
    読み込みに失敗しました（{{_fetch?.statusCode}}）。
    <button data-click-fetch="..." data-click-bind="#result"
            data-click-fetch-state="#result">再取得</button>
  </div>
  <ul data-if="_fetch?.success" data-each="items"><li>{{name}}</li></ul>
</div>
