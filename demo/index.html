12. history.pushState デモ

data-click-history で履歴 URL を更新し、フォーム値やクエリパラメータを反映します。

13. フェッチ状態の宣言的参照 (data-fetch-state)

data-fetch-state でフェッチの読み込み中・成功・失敗を _fetch として参照し、data-if だけでエラー表示や再取得導線を出し分けます。

data-fetch-state デモを開く

14. 同一 name ラジオの data-change-bind

同じ name のラジオを data-change-bind で別要素の1キーへ書き、選択値で依存欄を data-if 出し分け。前後どちらの遷移でもスカラ更新・即時再評価されます。

同一 name ラジオ デモを開く

15. data-if 非表示分岐はフォーム送信対象外

同じ name の入力を data-if で出し分けても、非表示分岐（data-if-false）の値はフォーム値収集の対象外。表示中の分岐の値だけが送信されます。

data-if 送信除外 デモを開く

16. <title> への実行時バインド

<head> もスキャン対象なので、<title> 自身に data-fetch / data-bind を付ければテキストの {{}} 補間が効きます。タブのタイトルに会社名を実行時取得で反映します。
